Beyond Mankind | Brytenwalda Studios

Beyond Mankind is an action RPG, based off the novel Evil Genes.

I joined the project late 2018 as an environmental artist, for the role I had to design all the visual compoents outdoor scenes. I also programmed certain environmental features such as cave lighting systems. I was also tasked with the optimisation of certain scenes which involved analysing the CPU and GPU performance using tools such as the Unity profiler.

The following tools were used:

  • Unity3D
  • Visual Studio
  • C#
  • Blender
