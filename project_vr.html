Lenses VR

Prototype application that simulates different eye conditions in VR.

This application was apart of a freelance project in which I worked alongside a QIE surgeon who wanted to prototype the use of VR to demonstrate different eye conditions. The project consists of a management section in which custom lens effects can be created. Different lenses can be loaded in the experience section in which a lens preset(s) and an environment are selected to view in VR. The different lenses and be toggled in VR to see the different effects.

The project was completed in approximately four months

The following tools were used:

  • C#
  • Visual Studio
  • Unity3D
  • Steam VR API
  • Blender
