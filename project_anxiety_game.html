Anxiety: Revelation of Mind

Anxiety: Revelation of Mind is a survival horror game set during WW1.

This game was my first project for not only game development but programming in general. I immensely enjoyed developing this project and it is because of this that I ended up choosing a career path in computing after highschool. The entire game was created from scratch using Unity3D. The project involved many aspects ranging from character controllers, file systems, enemy AI, inventory systems and much more. The project took one year to complete (2015-2016).

The following tools were used:

  • C#
  • JSON
  • Visual Studio
  • Unity3D
  • Maya
  • Mixamo

You can find more information and also download the project on the game's itch.io page
