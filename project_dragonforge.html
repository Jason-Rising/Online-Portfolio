Dragon Forge | Legend Studios

Dragon Forge is a base-building adventure game about managing massive amounts of resources by foraging and crafting. Dragon Forge is currently still in development.

I joined this project in October 2019 as an environment designer, this role involves developing all visual aspects of the game whilst managing game performance.

The following tools were used:

  • Unity3D
  • Visual Studio
  • C#
  • Blender
  • TortoiseSVN
